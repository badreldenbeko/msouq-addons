Egyptian E-Invoice Integration Fix

The Main Idea

Fixing the issue that raise when a customer is individual and an invoice amount less than Invoicing Threshold.

How to raise this issue ?

  • create individual customer.
  • configure invoicing threshold as 50000.
  • create invoice with amount less than the amount configured in invoicing threshold.
